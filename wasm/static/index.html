Privacy Statement:

Your documents or Dropbox credentials will never be transferred to any Bluepaper server. Bluepaper runs directly on your own computer using web technologies. If you grant Bluepaper access to your Dropbox account then your browser will download a list of recently modified Paper documents from Dropbox directly to your computer without going through any intermediary. If you check the box labeled "remember me" then Bluepaper will store a small access token in a so-called cookie in your browser so that you don't have to grant access again the next time you use Bluepaper. Uncheck that box if you are using a shared computer or if you don't consent to cookies. If you left the box checked unintentionally, don't worry: you can delete the cookie and revoke access permissions at any time.
